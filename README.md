# 💰 Crypto Price Checker

A simple and fast **live cryptocurrency price checker** built with Python using the CoinLore API.
This project fetches real-time prices of the top 100 cryptocurrencies and allows users to search by **coin name or symbol**.

---

## 📌 Features

- 🔍 Search crypto by name or symbol (e.g. `bitcoin`, `BTC`)
- 📊 Live price from CoinLore API
- ⚡ Fast and lightweight (no API key required)
- 🧠 Beginner-friendly Python project

---

## 🛠️ Tech Stack

- Python 3
- Requests library
- CoinLore API

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/PiMoOX/crypto-price-checker.git
cd crypto-price-checker
```

Install the dependencies:

```bash
pip install -r requirements.txt
```

---

## 🚀 Usage

Run the script:

```bash
python main.py
```

Then enter a coin name or symbol when prompted (e.g. `bitcoin` or `BTC`) to see its live price.
